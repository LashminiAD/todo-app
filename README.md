# 📝 Offline To-Do List App

A simple and efficient **To-Do List Web App** built using HTML, CSS, and JavaScript that stores tasks in the browser using localStorage.

👉 Your tasks remain saved even after refreshing or reopening the page.

---

## 🚀 Live Demo

🔗 [https://LashminiAD.github.io/todo-app/](https://lashminiad.github.io/todo-app/)

---

## ✨ Features

- ➕ Add new tasks  
- ✅ Mark tasks as completed  
- 🗑️ Delete tasks  
- 💾 Data persistence using localStorage  
- 🔄 Tasks remain even after page refresh  
- 📱 Simple and responsive UI  

---

## 🛠️ Tech Stack

- HTML5 – Structure  
- CSS3 – Styling  
- JavaScript (Vanilla JS) – Functionality  
- localStorage API – Data persistence

## 🧠 How It Works

- Tasks are stored in the browser using localStorage  
- On page reload, tasks are fetched and displayed  
- Completed tasks are saved persistently
